# job4j_di

В проекте реализован простой DI-контейнер на Java.

DI — Dependency Injection, внедрение зависимостей.

Контейнер `Context`:

- регистрирует классы;
- анализирует конструкторы через reflection;
- находит зависимости;
- создает объекты;
- хранит их в `Map`;
- отдает готовые объекты через `get()`.

## Технологии

- Java 21
- Maven
- JUnit 5
- AssertJ
- Checkstyle
